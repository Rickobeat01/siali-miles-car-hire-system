# Siali Miles Safaris Car Hire Management System

A professional desktop application for managing vehicle rentals at Siali Miles Safaris.

## Features

- Vehicle Management
- Customer Management
- Rental Management
- Payments & Expenses
- Monthly Reports
- PDF & Excel Reports
- Dashboard
- User Login

## Technology

- Python
- CustomTkinter
- SQLite
- ReportLab
- OpenPyXL
